# Sellflux Task Manager

## 🚀 Destaque Técnico

Esta aplicação segue rigorosamente os requisitos do desafio:

- Uso exclusivo de I/O assíncrono com callbacks nativos do Node.js
- Queries SQL puras e parametrizadas ($1, $2) utilizando o módulo pg
- Não utiliza Promises, async/await ou ORMs

## 🗂️ Organização do Código

O backend segue a estrutura exigida:

- routes/
- controllers/
- models/

Separando claramente responsabilidades da aplicação.
